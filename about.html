Courses I Learnt

Full Stack Web Development by Angela Yu on Udemy

I completed the Full Stack Web Development course by Angela Yu, which provided me with a comprehensive understanding of both front-end and back-end development. The course covered:

  • HTML, CSS, and JavaScript: Core web technologies for building interactive and responsive web pages.
  • Node.js and Express.js: Server-side development and API creation.
  • MongoDB: Database management and integration.
  • React.js: Building dynamic user interfaces and single-page applications.
  • Version Control with Git and GitHub: Collaborating on projects and maintaining code versions.

Machine Learning A-Z by Kirill Eremenko on Udemy

The Machine Learning A-Z course by Kirill Eremenko was a deep dive into the world of machine learning and artificial intelligence. The course covered:

  • Data Preprocessing: Cleaning and preparing data for analysis.
  • Regression Techniques: Simple Linear Regression, Multiple Linear Regression, and Polynomial Regression.
  • Classification Algorithms: Logistic Regression, K-Nearest Neighbors, Support Vector Machine, and Decision Trees.
  • Clustering: K-Means Clustering and Hierarchical Clustering.
  • Association Rule Learning: Apriori and Eclat algorithms.
  • Reinforcement Learning: Upper Confidence Bound and Thompson Sampling.
  • Natural Language Processing: Text classification and sentiment analysis.
  • Deep Learning: Artificial Neural Networks and Convolutional Neural Networks.
  • Model Selection and Boosting: Techniques to improve model performance.

Prompt Engineering by Julian Melanson

The Prompt Engineering course by Julian Melanson introduced me to the cutting-edge techniques of working with AI language models. The course covered:

  • Understanding AI Models: Basics of how AI language models work and their applications.
  • Crafting Effective Prompts: Techniques to design prompts that elicit the desired responses from AI models.
  • Fine-Tuning Models: Adapting pre-trained models for specific tasks.
  • Ethical Considerations: Ensuring responsible and ethical use of AI technologies.
  • Real-World Applications: Using prompt engineering in chatbots, content generation, and automated customer support.
